# ✈️ SkyRoute Analysis Dashboard

## 📌 Project Overview

The **Airlines Routes Analysis Dashboard** is an interactive Power BI report designed to analyze airline operations, route performance, aircraft utilization, financial metrics, and operational KPIs. The dashboard enables users to monitor airline performance through dynamic visualizations and DAX-based calculations.

---

## 🎯 Objectives

* Monitor airline operational performance.
* Analyze route efficiency.
* Evaluate aircraft utilization.
* Track financial performance.
* Provide actionable insights for decision-making.

---

## 🛠️ Tools & Technologies

* **Power BI Desktop**
* **DAX (Data Analysis Expressions)**
* **Power Query**
* **CSV Dataset**

---

## 📊 Dashboard Pages

### 1️⃣ Dashboard

Provides a high-level overview of the airline business.

**Key KPIs**

* Total Flights
* Total Revenue
* Total Routes
* Average Load Factor
* On-Time Performance
* Total Profit

---

### 2️⃣ Route Analysis

Analyze route performance using interactive visuals.

**Insights**

* Top Routes
* Most Popular Destinations
* Route Revenue
* Route Performance Comparison

---

### 3️⃣ Air Craft Analysis

Monitor aircraft performance and utilization.

**Insights**

* Aircraft-wise Flights
* Aircraft Utilization
* Revenue by Aircraft
* Fleet Performance

---

### 4️⃣ Financial Analysis

Track airline financial performance.

**Metrics**

* Total Revenue
* Total Cost
* Total Profit
* Profit Margin
* Revenue Trend
* Cost Analysis

---

### 5️⃣ Operation Overview

Monitor daily airline operations.

**KPIs**

* Flight Status
* On-Time Flights
* Delayed Flights
* Cancelled Flights
* Operational Performance
* Flight Distribution
